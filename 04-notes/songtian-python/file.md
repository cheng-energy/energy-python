# 文本
## 文件类型
- 文本文件（有统一编码UTF8）--txt，.py
- 二进制文件（很多都是二进制文件，视频等等等）

## 文件的打开与关闭
- 步骤---打开文件----操作文件-----关闭文件
### 打开
- open（文件名，打开模式）=变量名
- 文件名应当增加文件路径（注意反斜杠和斜杠使用）

- 打开模式（7种）（默认会以只读模式打开）

1. 'r',只读模式,文件不存在会报错
2. 'w',覆盖写模式，文件不存在则会创建，如果文件存在，原先存在的文件会被完全覆盖
3. 'x',创建写模式，文件不存在则会创建，存在会报错
4. 'a',追加写，如果文件存在会在文件后面追加内容，如果文件不存在则会创建文件
5. 'b',二进制方式打开文件
6. 't',以文本文件模式打开
7. '+'，与r/w/a一起使用，在原先功能基础上增加同时读写的功能
8. 还可以混合‘rt’文本模式只读打开。‘a+’文本形式，追加写模式可以读文件。‘wb’二进制形式可以覆盖写


### 文件的关闭
- 变量名.close

如果没有关闭文件只要程序正常退出文件也会关闭，不过尽量关闭文件




### 文件内容的读取
- <f>.read(size=-1)
- <f>.readline(size=-1)      # 读取某一行
- <f>.readlines(size=-1)    # 读取全部

文件全文处理
- 全部读入文件，然后进行处理（如果读取一个文本文件的全部，则对于所有内容进行处理）（大文件不可以）
- 使用上面的文件内容读取的第一个部分，增加一个while循环，只要读入的内容!=' '(读入内容不是空)，然后就进行while循环下方的代码块操作处理文件，最后记得关闭文件
- 按照行来读取（按照行来存储的文件），一次性全部读入文件，然后使用for循环,例如for line in 储存文件的变量.readlines（）:----------(一次读入，逐行处理）
- 分行读入，逐行处理，for line in 储存文件的变量


### 数据的文件写入
- <f>.write(s)---- 向一个文件写入几个字符串或者字节流
- <f>.writelines(lines)----将一个全为字符串的列表写入文件，不是以行写入，是将元素拼接写入
- <f>.seek(offset)----改变当前文件操作指针，（0就是文件开头，1是当前位置，2是文件结尾）offset那个位置是0或者1或者2
