## 双分支结构
- if-else
- 紧凑形式：<表达式1> if <条件> else <表达式2>

注意：
1. 紧凑形式并不支持带等号赋值的形式，以猜数字游戏为例：
```python
#简单猜数字游戏的二分支紧凑形式
guess = eval(input())
print("猜{}了".format("对" if guess == 44 else "错"))
```

## 程序异常的处理(try-except)
### try-except

使用结构：

try:
    <语句块1>
except：
    <语句快2>

说明：当程序运行时，首先运行语句快1，如果语句快1报错则执行语句快2，反之则不执行语句块2

### 指定异常类型的处理

try:
    <语句块1>
except 异常类型：
    <语句快2>

说明：只有语句块1发生指定的异常类型时才会执行语句块2（异常类型是python已经规定好的）

### 高级异常的处理（增加了else和finally）
try:
    <语句块1>
except：
    <语句快2>
else：
    <语句块3>
finally:
    <语句块4>

说明：语句块3是在不发生异常才会被执行。但是不管是否有异常，语句块4都会被执行
