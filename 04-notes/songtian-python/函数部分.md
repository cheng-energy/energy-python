# python的语法3
## 函数部分
### 函数的定义
```python
def 函数名称（函数参数）
    函数体代码
    return <返回值>
```
案例
```python
# 计算n的阶乘
def fact(n):
    s = 1
    for i in range(1,n+1):
        s *= 1
    return s
```
定义中的参数n计算占位符
### 函数的使用和调用过程
- 调用时给出实际参数，使用实际参数替换定义中的占位参数，调用后就会得到返回值
案例
```python
# 计算n的阶乘
def fact(n):
    s = 1
    for i in range(1,n+1):
        s *= i
    return s
a = fact(10)
print(a)
```

### 函数的参数传递
- 函数可以有参数也可以没有，但是必须要有括号

可选参数：

- 函数定义时可以为某些参数提供默认值，构成可选参数（可选参数就是调用函数时，某些变量我们不提供，只提供其他变量。这种可以提供也可以不提供的参数叫可选参数）
- 可选参数必须放在非可选参数后面（必选参数）--------一种强制约定
实例：
```python
#计算n的阶乘除m
def fact(n,m=1)         #提供m的数值时就按照提供的数值，不提供时就按照m=1默认
    s = 1
    for i in range(1,n+1)：
        s *= i
    return s//m
a = fact(5)
b = fact(5,5)
print(a)      #120
print(b)      #24
```

可变参数传递：
- 函数定义时可以设计可变数量变量参数（不确定参数的总数量）

    def <函数名>（<参数>，*b）
        <函数体>
        return  <返回值>

实例：
```python
#计算n阶乘和其他数相乘的结果
def fact(n,*b):
    s = 1
    for i in range(1,n+1):
        s *= i
    for j in b:
        s *= j
    return s
a = fact(5,2,2)
print(a)
```
** 一定要注意代码块的缩进 **
- 函数调用时，参数可以按照位置或者名称（例如n=10）来传递

### 函数的返回值
- 函数可以使用return也可以不用（也可以写一个return后面没有返回值）

返回多个结果：
“ return  s//m，n，m ”可以这样写，返回一个括号里面带着这三个数（元组类型）

## 局部变量和全局变量[程序全局变量，函数局部变量（定义中的变量）]
  
规则:
1. 局部变量和全局变量是不同变量(基本数据类型)
2. 局部变量如果是组合数据类型（如列表），且在函数定义中未被创建，则默认该组合数据类型是全局变量

- 可以使用global在函数定义时使用全局变量，案例如下
```python
n,s = 10,100
def fact(n):
    s = 1
    for i in range(1,n+1):
        s *= i
    return s               #局部变量s的值是3628800
print(fact(n),s)      #   3628800,100        全局变量s的值是100

---
n,s = 10,100
def fact(n)
    global s      #声明s是全局变量
    for i in range(1,n+1):
        s *= i
    return s
print(fact(n),s)    #362880000
```
- 组合数据类型案例如下
```python
ls = ['A','B']
def fact(a):
    ls.append(a)
    return ls
print(fact('c'))        #'A''B''c'
# ################
ls = ['A','B']
def fact(a):
    ls = []
    ls.append(a)
    return ls
print(fact('c'))         #'c'
```

## lambda函数
- 学习后的自我阐述：所谓lambda函数就是没有名字的函数，使用方法如下
- <函数名> = lambda <参数> : <表达式>---------这里是表达式而不是函数体
```python
f = lambda x,y :x+y
f(2,3)    #5
# -----------------------------------------------------------------------
a = lambda :'字符串'
print(a())    #字符串
# lambda的使用也可以没有参数，如上
```
使用注意事项
1. lambda函数一般在遇到比较复杂的函数时会使用，例如有一些函数的参数就是另一个函数
2. lambda还有一些固定的使用方式，老师建议逐步掌握
3. 一般情况下还是建议def定义函数

## 代码复用和函数递归
### 代码复用：
- 将代码通过封装抽象成资源实现（函数，对象等方式实现代码复用）
- 函数在代码层面建立了初步抽象，级别比较低
- 对象可以通过属性和方法对代码进一步抽象（面向对象编程）

模块化设计思想：（在代码复用基础上）
- 通过函数和对象将程序划分为模块（分而治之）
- 函数体内部要紧耦合（两个部分交流多无法独立存在），函数和其他代码（模块）之间尽可能松耦合。这样代码复用概率更高。

### 函数递归(数学归纳法思维在编程中的体现)
- 函数递归的理解：函数定义中，调用函数自身的方法就是递归（类似于计算n的阶乘）

 特征；
 1. 链条：计算过程中存在递归链条，类似于n的阶乘=n*n-1的阶乘
 2. 基例：存在着一个或者多个不需要递归的基例(类似与n=0时，与其他值不存在关系，他不需要递归)


- 函数递归（函数+分支语句）的调用过程，案例:
```python
#函数递归的调用，计算n的阶乘
def fact(n):
    if n == 0:
        return 1
    else:
        return n*fact(n-1)
fact(5)          #120
```
当n=5时返回的是5×fact（4），计算机接下来会把这个储存然后计算fact（4）的值就是4×fact（3），结合之前储存的就是5×4×fact（3），以此类推直到n=0的时候返回1最终计算结果为120

汉诺塔问题：
```python
# 汉诺塔问题：
"""有ABC三个柱子，在A柱子上有n个从上到下由小到大的圆环，规定小圆环只能放在大圆环上面，怎么能够让A上面的n个圆环放在C柱子上"""
count = 0 #记录搬运圆环的次数
def hannot(n,fir,sec,end):    #函数hannot有四个参数，n是圆环的数量，fir是圆环所在的初始柱子，sec是圆环过渡的中间柱子，end是圆环要放到的最终柱子
    global count
    if n == 1:
        print('{}:{}-->{}'.format(1,fir,end))
        count += 1
    else:
        hannot(n-1,fir,sec,end)
        print('{}:{}-->{}'.format(n,fir,end))
        count += 1
        hannot(n-1,sec,fir,end)     # 搬运了一个圆环还剩下n-1个圆环需要处理，此时n-1个圆环在B柱子上，搬运n-2个到A柱子上（n-2是指除了最下面那个柱子以外的柱子），然后把B柱子上最后一个圆环放在C柱子上成功实现目标。然后此时A柱子上有n-2个圆环，继续搬运
hannot(3,'A','B','C')
print(count)
```
函数的递归调用，关键是找到基例和链条。
