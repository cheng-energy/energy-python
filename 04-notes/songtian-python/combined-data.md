# 组合数据类型

## 集合
- 集合和数学上定义产不多，都具有唯一性和无序性

### 集合的创建
1. 直接使用{}来创建集合
2. 使用函数set()来创建集合
```python
a = set('python123')
print(a)             #{'n', '3', 'y', 'p', 't', '2', 'o', '1', 'h'}
```

### 集合的操作符（交，并，补，差）
1. 集合的并运算：S|T----返回一个新的集合，集合中内容是S和T中所有元素
2. 集合的对称差集运算:S^T----返回一个新的集合，集合中的内容是S和T中所有元素（除去S和T共有的元素）
3. 集合的差运算:S-T----返回一个新的集合，集合中的元素是S中的元素除去T中元素之后剩余的元素
4. 集合的交运算:S&T----返回一个新的集合，集合中的元素是同时在S和T中的元素
5. S <=T 或者S < T 等关系运算符，返回值是True或者False
6. 增值运算符：例如：S|=T----类似于+=运算，这种操作会修改S集合中的元素（就是S=S|T）
