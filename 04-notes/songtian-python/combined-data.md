# 组合数据类型
- 集合，元组，列表，字典，
## 集合
- 集合和数学上定义产不多，都具有唯一性和无序性

### 集合的创建
1. 直接使用{}来创建集合
2. 使用函数set()来创建集合
```python
a = set('python123')
print(a)             #{'n', '3', 'y', 'p', 't', '2', 'o', '1', 'h'}
```

### 集合的操作符（交，并，补，差）
1. 集合的并运算：S|T----返回一个新的集合，集合中内容是S和T中所有元素
2. 集合的对称差集运算:S^T----返回一个新的集合，集合中的内容是S和T中所有元素（除去S和T共有的元素）
3. 集合的差运算:S-T----返回一个新的集合，集合中的元素是S中的元素除去T中元素之后剩余的元素
4. 集合的交运算:S&T----返回一个新的集合，集合中的元素是同时在S和T中的元素
5. S <=T 或者S < T 等关系运算符，返回值是True或者False
6. 增值运算符：例如：S|=T----类似于+=运算，这种操作会修改S集合中的元素（就是S=S|T）

# 序列类型（一种基类类型）
- 一些有顺序的数据组成的类型，因为有顺序，所以元素可以相同
- 一般使用的是序列类型衍生的三种类型；字符串类型，元组类型。列表类型
- 序列类型存在正向递增的索引序号（从0开始）和反向递减的索引序号（从-1到-n），可以借助字符串来理解这种
- 字符串类型就是序列类型的扩展，可以理解每个字符是一个元素，字符之间有顺序，字符串还可以进行索引切片操作

## 序列处理函数以及方法
- 操作符（6个）
1. x in s -----返回True或者False，判断x是否在s中
2. x not in s -----返回True或者False，判断x是否不在s中
3. s + t ------连接两个序列s和t
4. s * n  或 者  n * s------表示把序列s复制n次
5. s[  i  】 ------索引，返回s中的第i个元素，i是序列的序号（从0开始）
6. s【i:j】或s【i；j；k】----切片操作，返回序列s中从第i到第j以k为步长的元素子序列
```python
ls = ['python',12,'吃饭']
ls[::-1]  #[ '吃饭', 12, 'python']
```
## 序列通用的函数和方法（5个）
1. len(s)---返回序列s的长度
2. max(s)---返回序列s中最大值
3. min(s)---返回序列s中最小值
4. s.count(x)---返回序列s中x出现的总次数
5. s.index(x)或者s.index(x,i,j)---返回xuls从i到j第一次出现元素x的位置（返回一个序号）

PS：进行max和min等运算时注意比如整数类型和字符串类型不能比较，如果想返回字符串中最大的值，字符之间比较按照字母序来比较
