# 关于语音识别功能
- 使用vosk
#### 测试阶段使用最小的模型开始下载
- vosk模型官网：https://alphacephei.com/vosk/models
```py
import json  # 处理json文件
import queue    # 创建一个新的线程，避免数据冲突
import sys   # 系统的相关功能，在系统加载失败的时候强制关闭程序
import sounddevice as sd   # 获取系统麦克风的使用权限，实时使用麦克风录制音频数据
from vosk import Model,KaldiRecognizer  # 准备识别的环境，执行语言识别
```
- 使用json库来处理vosk返回的识别结果（识别结果是一个json文件）
- queue 会把后台的音频数据放在线程里，有.put方法放置数据，.get方法读取数据
- sys 则是使用系统的相关操作，当程序出现异常时，强制退出程序而不是报错
- import sounddevice as sd 获取电脑的麦克风权限
