# 网络机器人（爬虫）
## robots协议

`robots协议是一种君子协议`，包含内容如下
- user-agent： 用户代理，通过请求头来确定爬虫的类型
- disallow： 禁止爬取的内容
- allow： 允许爬取的内容
- sitmeap： 网络地图，帮助爬虫爬取
- crawl-delay： 访问的间隔时间

打开方式：
- 域名后加/robots.txt

##### 简单了解前端语言
- 由于这个项目爬取回来的时候是前端的代码，所以要进行数据的清洗，需要了解一定的知识
1. 前端代码由HTML,CSS,Javascript组成
2. HTML 是超文本语言，主要展示的就是内容，也是我们最需要注意的部分，他由以下部分组成
    - <标签名，key= valus>这个是开始标签，</标签名>这个是结束标签（可以嵌套使用）--其中key=value是指标签的属性
    - 每一个标签名都有固定的作用，例如<img>就是用来展示图片的
3. CSS是主要负责页面的美好
4. JS 是负责交互的效果
