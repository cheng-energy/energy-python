# 网络机器人（爬虫）
## robots协议

`robots协议是一种君子协议`，包含内容如下
- user-agent： 用户代理，通过请求头来确定爬虫的类型
- disallow： 禁止爬取的内容
- allow： 允许爬取的内容
- sitmeap： 网络地图，帮助爬虫爬取
- crawl-delay： 访问的间隔时间

打开方式：
- 域名后加/robots.txt
