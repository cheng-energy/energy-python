# anaconda 的命令
- conda create -n 虚拟环境的名字 python=python版本  # 创建虚拟环境
- conda activate 虚拟环境的名字 #进入虚拟环境
- conda deactivate 虚拟环境的名字     #突出虚拟环境
- conda remove -n 虚拟环境的名字 --all   # 删除虚拟环境
- conda list env(conda env list 显示更精确)
- 使用命令行可以大大减少进入沙箱的时间
