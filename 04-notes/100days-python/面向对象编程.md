# 面向对象
## 面向对象基础
- 面向过程编程：把大的流程细化为不同的小步骤，然后依次执行步骤最终实现任务（简单线性的任务）
- 面向对象编程：
1. 万物皆对象，具体的人或物的数字化身。
2. 把一个人或者一个物的特征（属性）和功能（方法）打包在一起，是面向对象编程的基本单元
- 面向对象编程的类
1. 类是指具有相同属性和方法的模板
- 面向对象编程的对象
1. 对象是类的实例，是基于类创建的

**对象是由类创建的，创建对象的过程也叫类的实例化，一个类可以创建无数个对象）**


## 类与对象
### 类名的命名规范

**类名遵循大驼峰命名法，每个单词首字母大写，单词间没有分隔符**
#### 不推荐的创建方法
```python
# 定义类
# class 类名:
#     pass

# 创建对象
# 对象名= 类名（）
# 对象名.属性名1= 属性值1
# 对象名2.属性名2= 属性值2
class Car:
    pass
c1 = Car()
c1.color = "red"  # 添加属性是动态的
c1.brand = 'X5'
c1.price = 5000000
print(c1.__dict__)
print(c1)   
#<__main__.Car object at 0x000001AD835574D0>内存地址，每次运行程序内存都会改变
# __dict__ 是python自定义实例中的一个特殊属性，以字典形式储存对象的属性
```
`不建议这种方法创建，因为动态的修改会降低代码的可读性增加开发难度，类并没有起到模板作用`

#### 推荐创建方法
```python
# 定义类
# class 类名:
#     def __init__(self,参数列表) #__init__初始化方法，对象创建后自动调用，主要用于设置对象的初始状态（对象的属性），self表示当前创建的实例对象
#         self.属性名=参数值
        
# 创建对象
# 对象名=类名（参数列表）#参数列表中参数的数量要和def的参数列表中的数量一致
```
`def在类外是定义函数，在类中是称为方法`

```python
class Car:
    def __init__(self, color,brand,price):
        self.color = color
        self.brand = brand
        self.price = price
        print('初始化完毕')
# 创建对象
c1 = Car('red','xmw','50000')
print(c1.__dict__)
#初始化完毕 ---说明创建对象的时候会自动调用那个类
# {'color': 'red', 'brand': 'xmw', 'price': '50000'}
```
## 实例方法
- 任何为类的实例对象定义方法
```python
class Car:
    def __init__(self, name,color,price):
        self.color = color
        self.name = name     #self.属性名是实例绑定属性的唯一方式，只有在类中这样绑定，后续对象才知道有这样的方法
        self.price = price
        print('初始化完毕')
    def running (self):
        print(f'{self.color} 的{self.name} 正在行驶')
    def total_price(self,discount):
        return self.price * discount
# 创建对象
c1 = Car('CHX','red',50000) # 这种数值注意不带引号，如果带引号成为字符串后续计算可能有问题
Price = c1.total_price(0.5)
print(Price)
c1.running()
# 25000.0
# red 的CHX 正在行驶
```

## 魔法方法
- 在python中提供以双下划线开头和结尾的特殊方法，用于定义类的特殊行为，例如`__init__`
- 魔法方法不需要我们手动调用，python解释器会在合适时机自动调用

|魔法方法|描述|
|-------|--------|
|`__init__`|初始化的方法|
|`__str__`|字符串表示的方法|
|`__eq__`|比较两个对象是否相等（equal）|
|`__lt__`,`__le__`,`__gt__`,`__ge__`|支持比较两个对象的大小（小于（less than），小于等于（less than or equal），大于（greater than），大于等于（greater than or equal）|

```python
class Car:
    def __init__(self, name,color,price):
        self.color = color
        self.name = name     #self.属性名是实例绑定属性的唯一方式，只有在类中这样绑定，后续对象才知道有这样的方法
        self.price = price
        print('初始化完毕')
    def running (self):
        print(f'{self.color} 的{self.name} 正在行驶')
    def total_price(self,discount):
        return self.price * discount
# 创建对象
c1 = Car('CHX','red',50000)
c2 = Car('CHX','red',50000)# 这种数值注意不带引号，如果带引号成为字符串后续计算可能有问题
print(c1)
print(c1==c2)
print(c1<=c2) # 报错,地址无法比较
# 如何才能比较呢，可以借助魔法方法
```
