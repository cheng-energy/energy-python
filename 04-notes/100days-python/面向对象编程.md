## 类的创建
- 我们可以使用class加上类名来创建类。
- 通过缩进，确定类的代码块，我们需要写一些函数，写在类中的函数我们通常称为方法（方法就是对象的行为，也就是对象可以接收的消息。方法的第一个参数通常都是self，它代表了接收这个消息的对象本身。）
```python
class Student:

    def study(self, course_name):
        print(f'学生正在学习{course_name}.')

    def play(self):
        print(f'学生正在玩游戏.')
```

## 创建和使用对象
- 定义好类以后，可以使用构造器语法来创建对象
```python
stu1 = Student()
stu2 = Student()
print(stu1)    # <__main__.Student object at 0x10ad5ac50>
print(stu2)    # <__main__.Student object at 0x10ad5acd0> 
print(hex(id(stu1)), hex(id(stu2)))    # 0x10ad5ac50 0x10ad5acd0
```
在类的后面加上括号就是所谓的构造器语法，返回的是输出内容在内存中的地址（16进制），我们定义的变量其实保存的是一个对象在内存中的逻辑地址（位置），通过这个逻辑地址，我们就可以在内存中找到这个对象。所以**stu3 = stu2** 这样的赋值语句并没有创建新的对象，只是用一个新的变量保存了已有对象的地址。


接下来就是给对象发消息，就是调用对象的方法，有两种方式
```python
# 通过“类.方法”调用方法
# 第一个参数是接收消息的对象
# 第二个参数是学习的课程名称
Student.study(stu1, 'Python程序设计')    # 学生正在学习Python程序设计.
# 通过“对象.方法”调用方法
# 点前面的对象就是接收消息的对象
# 只需要传入第二个参数课程名称
stu1.study('Python程序设计')             # 学生正在学习Python程序设计.

Student.play(stu2)                      # 学生正在玩游戏.
stu2.play()                             # 学生正在玩游戏.
```
在刚才定义的类student中，只有行为，没有属性，我们还可以使用 `_int_` 的方法来初始化属性，例如：
```python
'''面向对象编程'''
class student:
    def __init__(self,name,age):
        self.name = name
        self.age = age
    def study(self,course_name):
        print(f'{self.age}岁的{self.name}正在学习{course_name}')
    def play(self):
        print(f'{self.age}岁的{self.name}正在玩游戏')
stu1 = student('苏',18)
stu2 = student('程',19)
stu1.study('数学，语文，物理，化学，政治，英语')
stu2.play()
```
---
- 面向对象编程，对象的属性一般会被设置为私有（private）和受保护的（protect），这些属性是不可被直接访问的
- 对象的方法都是公开的，公开的方法是对象能获取信息，暴露给外界的接口，这就是访问可见性。
- 在 Python 中，可以通过给对象属性名添加前缀下划线的方式来说明属性的访问可见性，例如，可以用`__name`表示一个私有属性，`__name`表示一个受保护属性
```python
class Student:

    def __init__(self, name, age):
        self.__name = name
        self.__age = age

    def study(self, course_name):
        print(f'{self.__name}正在学习{course_name}.')


stu = Student('王大锤', 20)
stu.study('Python程序设计')
print(stu.__name)  # AttributeError: 'Student' object has no attribute '__name'
```
这个代码就会报错，但是下面的代码不会
```python
class student:
    def __init__(self,name,age):
        self.__name = name
        self.__age = age
    def study(self,course_name):
        print(f'{self.__age}岁的{self.__name}在学习{course_name}')
    def play(self):
        print(f'{self.__age}岁的{self.__name}在打游戏')
sty1 = student('程',19)
sty1.study('高等数学')
```
这是因为第一部分代码是在类的外部访问该属性，而第二部分是通过self.__name等在类的内部来访问属性，可以通过这种方法来访问

### 动态属性
- python是动态语言，就是运行时可以改变其结构，例如新的函数、对象、甚至代码可以被引进，已有的函数可以被删除或是其他结构上的变化”

在python中，我们可以动态增加对象的属性。
需要注意的是，对象的方法其实本质上也是对象的属性，如果给对象发送一个无法接收的消息，引发的异常仍然是AttributeError
- 例如，上面的代码增加`sty1.sex = '男'`,这就是为对象sty1增加了sex这个属性


如果不希望在使用对象时可以增加对象的属性，可以使用`__slots__`。
- 可以使用`__slots__('name','age')`，这样以后类中的studet对象只可以有name和age两个属性
```python
class student:
    __slots__ = ['__name','__age']
    def __init__(self,name,age):
        self.__name = name
        self.__age = age
    def study(self,course_name):
        print(f'{self.__age}岁的{self.__name}在学习{course_name}')
    def play(self):
        print(f'{self.__age}岁的{self.__name}在打游戏')
sty1 = student('程',19)
sty1.sex = '男'
```
这个就会报错，显示student这个类中的属性sty1没有sex这个属性，还有no `__dict__ `for setting new attributes


### 静态方法和类方法
