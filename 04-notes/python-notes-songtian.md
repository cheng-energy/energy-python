# python的语法3
## 函数部分
### 函数的定义
```python
def 函数名称（函数参数）
    函数体代码
    return <返回值>
```
案例
```python
# 计算n的阶乘
def fact(n):
    s = 1
    for i in range(1,n+1):
        s *= 1
    return s
```
定义中的参数n计算占位符
### 函数的使用和调用过程
- 调用时给出实际参数，使用实际参数替换定义中的占位参数，调用后就会得到返回值
案例
```python
# 计算n的阶乘
def fact(n):
    s = 1
    for i in range(1,n+1):
        s *= i
    return s
a = fact(10)
print(a)
```

### 函数的参数传递
- 函数可以有参数也可以没有，但是必须要有括号

可选参数：

- 函数定义时可以为某些参数提供默认值，构成可选参数（可选参数就是调用函数时，某些变量我们不提供，只提供其他变量。这种可以提供也可以不提供的参数叫可选参数）
- 可选参数必须放在非可选参数后面（必选参数）--------一种强制约定
实例：
```python
#计算n的阶乘除m
def fact(n,m=1)         #提供m的数值时就按照提供的数值，不提供时就按照m=1默认
    s = 1
    for i in range(1,n+1)：
        s *= i
    return s//m
a = fact(5)
b = fact(5,5)
print(a)      #120
print(b)      #24
```
